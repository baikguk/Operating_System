운영체제
 컴퓨터의 하드웨어와 소프트웨어를 총체적으로 관리하는 소프트웨어

컴퓨터 기본 구성 요소
 입력장치, 출력장치, 처리장치, 저장장치

메인프레임 컴퓨터 : 은행, 항공사등 대규모 사업장에서 사용
	터미널을 통해 접속
	시분할(Time sharing) 방식 : 컴퓨터 처리 능력을 시간 단위로 분할해 사용자에게 분할

슈퍼컴퓨터(HPC(High performance computer :메인프레임 컴퓨터로 부족한 처리능력을 보안하기 위해 사용됨, 기상예측, 통신망 설계, 석유 탐사등

클라우드 컴퓨팅 : 고성능 컴퓨터를 원격으로 사용
	고성능 컴퓨터의 능력을 분할하여 한 대의 물리적인 컴퓨터를 사용자들이 공유하여 사용함 -> 가상머신
		ㄴ 해당 기술을 가상화 기술이라 함


개인용, 휴대용 컴퓨터
	워크스테이션 : 소규모의 사용자가 공동으로 사용하기 위한 컴퓨터
	서버 : 네트워크로 연결된 다른 컴퓨터에 서비스를 제공하는 일종의 소프트웨어

임베디드 컴퓨터
	특수한 임무를 위한 컴퓨터 (가전, 군사)
	프로그램이 이미 저장됨 -> 새로운 프로그램 탑재 불가
